# Manual del producto Nubis R7

## Descripción

El **Router Nubis R7** es un dispositivo de conectividad para el hogar y pequeñas oficinas. Este documento contiene información interna que no está disponible en fuentes públicas.

## Especificaciones técnicas

- **Modelo interno**: NUB-R7-2024
- **Código de producto**: 88X-NUBIS-R7
- **Puertos**: 4 LAN Gigabit + 1 WAN. Puerto USB 3.0 para almacenamiento o impresora.
- **Alcance WiFi**: hasta 120 m² en condiciones estándar.
- **Frecuencias**: 2.4 GHz y 5 GHz (dual band).

## Códigos de error

| Código | Significado | Acción recomendada |
|--------|-------------|--------------------|
| E-204  | Tarjeta SD llena (en funciones de backup de configuración) | Borrar grabaciones o usar SD de mayor capacidad. Mínimo 8 GB recomendado. |
| E-301  | Sin conexión WAN | Comprobar cable Ethernet al router superior. Reiniciar el equipo manteniendo el botón de reset 10 segundos. |
| E-102  | Sensor de señal débil | Revisar ubicación del router; alejar de obstáculos metálicos. |

## Procedimiento de reinicio

Para reinicio completo: mantener pulsado el botón **RESET** (receso lateral) durante **10 segundos** hasta que el LED de alimentación parpadee en rojo tres veces. La configuración por defecto se restaura (SSID: NubisR7, sin contraseña; acceder a 192.168.1.1 para reconfigurar).
